# greedkod (ГридКод)
Сервис поиска промокодов для магазинов в России

# Стек
- PHP 8.4
- MySQL 9.5.0

# О проекте
На проекте гридкод размещаются только выгодные промокоды, без кучи мусорных псевдо-акций, без кучи переадресаций. Например, на сайте можно найти промокод для самоката и прочих сервисов
